# YouTube Playlist Duration

A browser extension that shows the **total duration** of all videos in a YouTube playlist — right on the playlist page!

No more manual calculations. Just open any playlist and instantly see the full length.

## Features

- Works on every YouTube playlist (sidebar panel & classic view)
- Automatically updates when you switch playlists
- Lightning fast, no external requests
- Supports hours, minutes and seconds (e.g. `3h 24m 15s • 47 videos`)

## Installation

1. Download or clone this repository
2. Open `chrome://extensions/` (or `edge://extensions/`, Brave, Opera etc.)
3. Enable **Developer mode** (top right)
4. Click **"Load unpacked"**
5. Select the folder with the extension
6. Done! Open any YouTube playlist — total duration appears automatically
